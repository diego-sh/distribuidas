Interfaces y Clases de Acceso a Datos (DAO) genéricas. La interfaz GenericDAO es la encargada de definir todos los métodos que serán implementados por todas las clases DAO. Por defecto se ha creado la clase DefaultGenericDAOImpl que es la que implementa todos estos métodos de forma genérica, todos las clases DAO a ser creadas deberán extender necesariamente de esta clase.
